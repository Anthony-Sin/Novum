name: "available-work-phases"
---
* Audit Planning: Use this Work Phase to receive and inventory a new submission package. The Submission Coordinator agent performs a CTD completeness check, maps all submitted documents to their CTD module, identifies missing required elements, and produces a sequenced Review Plan that governs all subsequent Work Phases. This is ALWAYS the first Work Phase for any new submission. Do NOT use this phase to perform clinical, statistical, or safety analysis — it produces only the Review Plan and document inventory.
* Background Research: Use this Work Phase to establish factual grounding before clinical analysis begins. Fetches the ClinicalTrials.gov registration records for all pivotal trials to compare against submitted protocols, retrieves current FDA guidance documents applicable to the indication and drug class, researches safety precedent in Drugs@FDA for the drug class, and verifies the sponsor's unmet medical need claims against published literature. Returns a structured Background Dossier that all subsequent clinical WorkPhases reference.
* Biostatistical Review: Use this Work Phase to independently recalculate all primary and key secondary efficacy endpoint statistics using the Biostatistical Recalculation Tool and Statistical Script Execution Tool. Audits for P-hacking, multiple comparisons violations, analysis population switching, and misleading effect size presentations. Returns a Statistical Audit Report with recalculated values, discrepancy verdicts, and a P-hacking Risk Score for every endpoint. If the task requires deep dataset investigation without a known statistical hypothesis, use Clinical Data Review instead.
* Clinical Data Review: Use this Work Phase to interrogate submitted SDTM and ADaM datasets for integrity, internal consistency, and protocol deviation impact using the Clinical Data Reconciliation Tool. Produces a Dataset Integrity Report with subject accountability tables, anomaly logs, missing data impact assessments, and protocol deviation summaries. Use this phase for dataset-level investigation; use Biostatistical Review for endpoint-level statistical recalculation.
* Safety Review: Use this Work Phase to systematically scan adverse event datasets, SAE narratives, and laboratory safety data for hidden or inadequately characterized safety signals using the Pharmacovigilance Signal Detection Tool. Produces a Safety Signal Report with TEAE incidence tables, a Hidden Signal Index, a Laboratory Safety Shift Table, and an overall Safety Signal Severity Rating. This phase must always precede the Benefit-Risk Determination phase.
* Regulatory Compliance Review: Use this Work Phase to audit submission documents against CFR Title 21, ICH guidelines, and current FDA guidance using the CFR Title 21 Compliance Audit Tool. Produces a Compliance Audit Report with a section-by-section compliance matrix, a numbered deficiency list, and an overall Regulatory Compliance Rating. Use this phase for procedural and labeling compliance assessment independent of clinical or statistical findings.
* Benefit-Risk Determination: Use this Work Phase to synthesize completed Biostatistical Review, Safety Review, and Clinical Data Review findings into a structured patient-centered risk-benefit analysis using the Risk-Benefit Ethics Analysis Tool. Produces a Benefit-Risk Report with absolute benefit and harm tables, an NNT/NNH comparison matrix, a clinical meaningfulness assessment, and an overall Benefit-Risk Determination. This phase may only be initiated after Biostatistical Review and Safety Review are both complete.
* Evidence Synthesis: Use this Work Phase to assemble and reconcile findings from multiple completed WorkPhases into a unified evidence summary using the Cross-Source Evidence Synthesis Tool. This phase bridges individual audit outputs and the final CRL draft. It is mandatory for any submission with more than two completed analytical WorkPhases. Produces a Synthesized Evidence Base document that serves as the structured input to the CRL Drafting phase.
* CRL Drafting: Use this Work Phase to draft the Complete Response Letter or Approval Recommendation document using the Complete Response Letter Drafting Tool. The Medical Writer agent compiles all WorkPhase findings — referenced by phase ID — into a formal, citation-specific regulatory document. This is ALWAYS the final analytical Work Phase before the Orchestrator issues a RETURN signal. May only be initiated after Evidence Synthesis is complete.
* Validation: Use this Work Phase ONLY to verify that the Complete Response Letter or Approval Recommendation produced by the CRL Drafting phase is internally consistent, that every deficiency finding is traceable to a specific WorkPhase output, and that the overall verdict is logically supported by the documented evidence. Do NOT use this phase to perform new clinical, statistical, or safety analysis. Returns a Validation Report confirming the CRL is ready for RETURN or listing specific inconsistencies requiring correction.
