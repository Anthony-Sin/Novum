* Paper Deconstruction Planning: Use this Work Phase to receive and inventory a new paper under investigation. The Paper Intake Coordinator performs a completeness check, maps all submitted materials to their components (main text, figures, supplementary data, preregistration records, data availability statements, author disclosures), identifies missing required elements, and produces a sequenced Investigation Plan that governs all subsequent Work Phases. This is ALWAYS the first Work Phase for any new paper. Do NOT use this phase to perform statistical, data integrity, or figure analysis — it produces only the Investigation Plan and document inventory.
* Registry and Provenance Research: Use this Work Phase to establish factual grounding before forensic analysis begins. Fetches the preregistration records (ClinicalTrials.gov, OSF, AsPredicted) for all studies to compare against the reported methods, retrieves current journal policies and COPE guidelines applicable to the paper type, researches author publication history and any prior retractions or expressions of concern on Retraction Watch and PubPeer, and verifies the authors' unmet knowledge gap claims against published literature. Returns a structured Background Dossier that all subsequent forensic WorkPhases reference.
* Statistical Reconstruction: Use this Work Phase to independently recalculate all primary and key secondary reported statistics using the Statistical Reconstruction Tool and Script Execution Tool. Audits for p-hacking, multiple comparisons violations, analysis population switching, and misleading effect size presentations. Returns a Statistical Audit Report with recalculated values, discrepancy verdicts, and a P-hacking Risk Score for every reported endpoint. If the task requires deep dataset investigation without a known statistical hypothesis, use Dataset Integrity Review instead.
* Dataset Integrity Review: Use this Work Phase to interrogate submitted or available raw data against reported summary tables and figures for internal consistency and integrity violations using the Dataset Integrity Tool. Produces a Dataset Integrity Report with subject accountability tables, anomaly logs, figure similarity assessments, and missing data impact analyses. Use this phase for dataset-level investigation; use Statistical Reconstruction for endpoint-level statistical recalculation.
* Adversarial Peer Review: Use this Work Phase to systematically scan reported figures, tables, and statistical scripts for hidden inconsistencies, impossible values, and internal contradictions using the Result Anomaly Detection Tool. Produces an Anomaly Signal Report with a list of flagged inconsistencies, impossible result citations, and an overall Anomaly Severity Rating. This phase must always precede the Research Impact Assessment phase.
* Publication Ethics Compliance Review: Use this Work Phase to audit the paper and its associated documentation against COPE guidelines, ICMJE authorship criteria, IRB requirements, data sharing mandates, and applicable journal policies using the Publication Ethics Compliance Tool. Produces a Compliance Audit Report with a section-by-section compliance matrix, a numbered violation list, and an overall Ethics Compliance Rating. Use this phase for ethical and procedural compliance assessment independent of statistical or data integrity findings.
* Research Impact Assessment: Use this Work Phase to synthesize completed Statistical Reconstruction, Adversarial Peer Review, and Dataset Integrity Review findings into a structured assessment of downstream harm caused by the paper using the Research Impact Assessment Tool. Produces an Impact Report quantifying citation-based downstream harm, clinical practice changes influenced by the paper, and wasted follow-on funding. This phase may only be initiated after Statistical Reconstruction and Adversarial Peer Review are both complete.
* Evidence Synthesis: Use this Work Phase to assemble and reconcile findings from multiple completed WorkPhases into a unified forensic evidence summary using the Cross-Source Evidence Synthesis Tool. This phase bridges individual audit outputs and the final recommendation draft. It is mandatory for any investigation with more than two completed analytical WorkPhases. Produces a Synthesized Evidence Base document that serves as the structured input to the Retraction Recommendation Drafting phase.
* Retraction Recommendation Drafting: Use this Work Phase to draft the Retraction Recommendation or Expression of Concern document using the Expression of Concern Drafting Tool. The Forensic Science Writer compiles all WorkPhase findings — referenced by phase ID — into a formal, citation-specific integrity document. This is ALWAYS the final analytical Work Phase before the Orchestrator issues a RETURN signal. May only be initiated after Evidence Synthesis is complete.
* Final Integrity Validation: Use this Work Phase ONLY to verify that the Retraction Recommendation or Expression of Concern produced by the Retraction Recommendation Drafting phase is internally consistent, that every integrity finding is traceable to a specific WorkPhase output, and that the overall verdict is logically supported by the documented forensic evidence. Do NOT use this phase to perform new statistical, data, or figure analysis. Returns a Validation Report confirming the recommendation is ready for RETURN or listing specific inconsistencies requiring correction.
